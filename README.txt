CIDL is a parser for a simple IDL-like language that is used in:
  https://github.com/Zeex/sampgdk

Supported things:
 - function declarations
 - constant declarations (integral, floating-point and strings)
 - function and parameter attributes
 - comments

CIDL needs Python 2.x and PLY:
  https://github.com/dabeaz/ply

Inspired by Mozilla's XPIDL:
  https://developer.mozilla.org/en/docs/XPIDL
